# Specs Release Notes

## Initial Public Release Design (000)

- Added the iOS-first public release product design for the family-managed medication adherence model.
- Defined the minimum production architecture, caregiver/patient role split, elderly-friendly UI direction, pricing policy, and scaling roadmap.
- Clarified that the initial release keeps patient mode free while caregiver mode owns medication, schedule, inventory, and premium management.

## History Schedule View (004)

- Added a History tab with a month calendar and day detail list.
- Status dots indicate taken (green), missed (red), pending (gray), with an always-visible legend.
- Caregivers see an empty state and action when no patient is selected.
- Full-screen updating overlay blocks interaction during loads and supports retry.
